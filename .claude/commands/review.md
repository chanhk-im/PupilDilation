# 코드 리뷰

아래 파일을 이 프로젝트의 컨벤션 기준으로 리뷰해줘.
코드를 직접 수정하거나 재작성하지 말고, 피드백만 제공해.

## 체크 항목

## 핵심 컨벤션
- 컨벤션은 노션 코드 컨벤션 페이지를 반드시 읽고 적용할 것
- 코드 컨벤션 페이지: `32d0e39821a680adbf60d7a8538d71a2`

**예외 처리**
- [ ] 커스텀 예외 사용 여부
- [ ] GlobalExceptionHandler와 에러 코드 일관성

**동시성 (ReservationService 해당 시)**
- [ ] 락 해제가 finally 블록에 있는지
- [ ] 트랜잭션 커밋 후 락 해제 순서가 맞는지
- [ ] 좌석 ID 오름차순 정렬 후 락 획득하는지

## 출력 형식

문제 있는 항목만 아래 형식으로 출력해:

**[심각도: 높음/중간/낮음]** `파일명:라인`
- 문제: ...
- 이유: ...
- 방향: ... (코드 작성 금지, 방향만)

설명할 만한 설계 포인트가 있으면 마지막에 따로 언급해줘.
